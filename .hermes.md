# HiggsfieldAgent

HiggsfieldAgent is a Hermes skill that receives an inspiration image from Telegram,
generates 4 FUFU character variations via the Higgsfield direct HTTP API,
downloads the output images, and sends them back to the user on Telegram.

## Key Paths

- `skills/higgsfield-fufu/` — Hermes skill definition (source of truth)
- `scripts/higgsfield_api.py` — direct HTTP API client (auth → upload → generate → poll → download)
- `scripts/get_aspect_ratio.py` — helper to map image dimensions to Higgsfield aspect ratios
- `config/hermes-config-snippet.yaml` — merge into `~/.hermes/profiles/higgsfield/config.yaml`

## Repo Location

This repo lives at `~/HiggsfieldAgent` in WSL Ubuntu. All skill paths and
script paths are relative to that WSL-native location.

## Instructions for Hermes

- **Do not create, edit, or delete skills.**
- **Do not call `skill_manage`.**
- The `higgsfield-fufu` skill is source-controlled in this repo and managed via git.
  Any changes to the skill workflow must go through a git commit, not in-session edits.
